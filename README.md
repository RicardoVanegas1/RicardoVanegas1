
# Sobre Mi

¡Hola! Soy Ricardo Arturo Vanegas Cadena, un Ingeniero Mecatrónico graduado de la Escuela Superior Politécnica del Litoral (ESPOL). Mi trayectoria profesional y académica está guiada por una profunda fascinación por la automatización industrial y la evolución hacia la Industria 4.0. Me apasiona conectar el mundo físico con el digital, transformando la industria tradicional a través de soluciones inteligentes, interconectadas y altamente eficientes.

Mi enfoque técnico se centra en el diseño eléctrico e informático de sistemas automatizados. Cuento con experiencia práctica en la programación de PLCs, la configuración de redes y protocolos industriales (como Modbus TCP y MQTT), y el desarrollo de arquitecturas de monitoreo en tiempo real mediante herramientas como Node-RED. Además, utilizo Python como una herramienta clave para el análisis de datos técnicos y el desarrollo de algoritmos de control avanzado.

Me considero una persona orientada a la excelencia técnica, estructurada y en constante aprendizaje. Mi objetivo es aportar al desarrollo tecnológico implementando sistemas SCADA dinámicos, soluciones de IoT Industrial (IIoT) y optimizaciones que impulsen la transformación digital en entornos productivos.

## 📂 Proyectos Destacados

### Sistema Automatizado de Adquisición de Datos para un Banco de Pruebas de Motores

Durante mi tesis de grado en la ESPOL, diseñé e implementé un sistema automatizado de adquisición de datos para un banco de pruebas de motores Hampden MPB-300.

El proyecto incluyó el diseño eléctrico e informático, la configuración e integración de:
* Un **PLC Siemens S7-300** como controlador central para el manejo de datos y la sincronización del proceso.
* **Analizadores de energía Siemens PAC3220** para medir, monitorear y evaluar variables eléctricas y el rendimiento energético.
* Un **backend y tablero de control en Node-RED** para construir una interfaz SCADA dinámica para la visualización en tiempo real y el registro de datos.
* Una **red de comunicación Modbus TCP** para garantizar una transferencia de datos confiable y de alta velocidad entre el hardware y la capa de supervisión.
